# ☁️ Azure Cloud & Infrastructure Security Portfolio
Welcome! I am a Quality Control Professional transitioning into **Azure Cloud Security and Administration**. This portfolio documents my technical skills, hands-on labs, and systematic approach to building secure, compliant enterprise cloud environments.

## 🛠️ Tech Stack & Focus Areas
*   **Cloud Platform:** Microsoft Azure (Entra ID, VNets, Azure Monitor, Defender for Cloud)
*   **Security Focus:** Zero Trust Architecture, Least Privilege Access, Configuration Compliance
*   **Core Methodologies:** Root-Cause Analysis, Auditing, Policy Enforcement

---

## 🔬 Hands-on Cloud Projects

### 🔒 [Lab 01] Secure Static Web Architecture
*   **Objective:** Deploy and isolate public-facing assets using automated enterprise guardrails.
*   **Services:** Azure Storage, Microsoft Entra ID, Azure Policy.
*   **QC & Compliance Check:** Successfully enforced an Azure Policy to block unencrypted storage creation and verified that anonymous public blob access was disabled at the root account level.
*   **[👉 View Lab Details & Architecture Diagrams](./labs/01-secure-storage/)**

### 🌐 [Lab 02] Isolated Network Topology with VNet Peering
*   **Objective:** Connect multi-tier infrastructure without exposing critical data nodes to the public internet.
*   **Services:** Azure Virtual Networks, VNet Peering, Network Security Groups (NSGs).
*   **QC & Compliance Check:** Verified data plane connectivity between isolated subnets while ensuring all inbound public RDP/SSH pathways were tightly restricted.
*   **[👉 View Lab Details & Architecture Diagrams](./labs/02-isolated-network/)**
